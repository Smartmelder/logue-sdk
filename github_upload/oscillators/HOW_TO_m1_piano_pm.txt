CONTROLS (NTS-1 mkII)
═══════════════════════════════════════════════════════════════

FRONT PANEL:
- TYPE knob: Select patch/preset
- A knob (SHAPE): HARDNESS (Hardness)
- B knob (ALT): DECAY (Decay Time)

PARAMETER MENU:
Hold OSC button + turn TYPE knob to select parameter
Adjust with A/B knobs

PARAMETERS:
0. HARDNESS - Hardness (Range: 0-1023, Default: 768)
1. DECAY - Decay Time (Range: 0-1023, Default: 614)
2. STIFF - Stiffness (Range: 0-1023, Default: 819)
3. DETUNE - Detune amount (Range: 0-1023, Default: 307)
4. BRIGHT - Brightness (Range: 0-1023, Default: 512)
5. BODY - Body resonance (Range: 0-1023, Default: 409)
6. CHORUS - Chorus amount (Range: 0-1023, Default: 256)
7. RELEASE - Release time (Range: 0-1023, Default: 205)
8. PRESET - Preset selection (Range: 0-7, Default: 0)
9. UNISON - Unison mode (Range: 0-2, Default: 1)

FEATURES
═══════════════════════════════════════════════════════════════

KORG M1 PIANO - HYBRID PHYSICAL MODELING

2-Op FM Exciter + Karplus-Strong + Stiffness Allpass
The authentic M1 metallic "clank" with wire inharmonicity

ARCHITECTURE:

1. EXCITER (2-OP FM BURST):
   - Carrier: Sine wave
   - Modulator: Sine wave @ 4.2:1 ratio (non-integer!)
   - High FM index (6-12) creates metallic "clang"
   - Ultra-fast envelope (5-20ms decay)
   - Creates iconic M1 attack transient

2. RESONATOR (EXTENDED KARPLUS-STRONG):
   - Delay line (string length)
   - Stiffness allpass filter (inharmonicity!)
   - Lowpass filter (damping)
   - Feedback control (decay time)
   
   STIFFNESS ALLPASS:
   - Introduces frequency-dependent delay
   - High frequencies travel faster (dispersion)
   - Creates "stiff wire" character

3. POST-PROCESSING:
   - Comb filter @ 2-3kHz (M1 DAC character)
   - Peaking EQ (body resonance)
   - Stereo chorus
   - Unison detune

PHYSICAL MODELING THEORY:
Piano string physics with inharmonicity factor B
Dispersion relation creates stretched partials (not perfect harmonics)
