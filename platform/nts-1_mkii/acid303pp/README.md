**SLIDE** - Portamento time

### Extended Parameters (Hold OSC)

**2. RESO** - Filter resonance

- Accent increases resonance

**3. ENV AMT** - Filter envelope amount
- How much envelope modulates filter

- 1ms - 1 second

- 2ms - 500ms
