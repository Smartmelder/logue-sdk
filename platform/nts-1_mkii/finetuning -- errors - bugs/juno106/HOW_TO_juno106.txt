MODULE TYPE: osc (Oscillator)

INSTALLATION:
Upload via KORG KONTROL Editor and select in OSC menu on NTS-1 mkII

═══════════════════════════════════════════════════════════════
CONTROLS (NTS-1 mkII)
═══════════════════════════════════════════════════════════════

FRONT PANEL:
- TYPE knob: Select patch/preset
- A knob (SHAPE): WAVE (Waveform)
- B knob (ALT): FX (Effect selection)

PARAMETER MENU:
Hold OSC button + turn TYPE knob to select parameter
Adjust with A/B knobs

PARAMETERS:
0. WAVE - Waveform selection (Range: 0-1023, Default: 341)
1. FX - Effect selection (Range: 0-1023, Default: 512)
2. PW - Pulse Width (Range: 0-1023, Default: 512)
3. DETUNE - Detune/Portamento amount (Range: 0-1023, Default: 409)
4. SUB - Sub oscillator level (Range: 0-1023, Default: 614)
5. HPF - High-pass filter (Range: 0-1023, Default: 102)
6. LFO - LFO Rate (Range: 0-1023, Default: 256)
7. LFODEP - LFO Depth (Range: 0-1023, Default: 307)
8. MODE - Mode selection (Range: 0-15, Default: 5)
9. FEAT - Features selection (Range: 0-15, Default: 2)

═══════════════════════════════════════════════════════════════
FEATURES
═══════════════════════════════════════════════════════════════

JUNO-106 MEGA EDITION - Complete recreation with ALL features

Complete Juno-106 synthesizer recreation including:
- Multiple waveforms (Saw, Square, Pulse)
- Chorus I & II effects
- Ring modulation
- Noise generator
- Sub oscillator
- High-pass filter
- LFO modulation
- Portamento/Glide
- Chord stacking
- 16-step sequencer
- Arpeggiator (8 patterns)
- Velocity sensitivity
- PolyBLEP anti-aliasing
