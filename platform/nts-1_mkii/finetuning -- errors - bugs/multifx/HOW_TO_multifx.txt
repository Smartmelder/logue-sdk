MULTIFX - MULTI-MODE MODULATION EFFECT
================================================================================

8 EFFECTS IN ONE UNIT!

================================================================================
OVERVIEW
================================================================================

MULTIFX is a multi-mode modulation effect optimized for:
- Techno production (chorus, flanger, phaser)
- House music (tremolo, autopan)
- Experimental (vibrato, ring mod)
- Live performance (mode switching)

FEATURES:
- 8 effect modes (Chorus, Flanger, Phaser, Tremolo, Vibrato, Autopan, Ring Mod, Combo)
- LFO modulation (rate, depth)
- Feedback control
- Tempo sync (6 divisions)
- LFO shapes (4 types)
- Stereo width control
- Color/tone filter
- Morphing between modes
- 10 PARAMETERS - MAXIMUM CONTROL!

Based on Korg logue SDK modfx template

================================================================================
8 EFFECT MODES
================================================================================

0: CHORUS - Classic chorus effect
1: FLANGER - Flanging with feedback
2: PHASER - Phase shifting
3: TREMOLO - Amplitude modulation
4: VIBRATO - Pitch modulation
5: AUTOPAN - Stereo panning
6: RING MOD - Ring modulation
7: COMBO - Multiple effects combined

================================================================================
10 PARAMETERS
================================================================================

0. MODE (TYPE knob when MOD held)
   Range: 0-7
   Default: 0 (CHORUS)
   Use: Select effect type
   - 0: CHORUS
   - 1: FLANGER
   - 2: PHASER
   - 3: TREMOLO
   - 4: VIBRATO
   - 5: AUTOPAN
   - 6: RING MOD
   - 7: COMBO

1. RATE (SHAPE knob)
   Range: 0-100%
   Default: 40%
   Use: LFO modulation speed
   - 0%: Very slow (0.5 Hz)
   - 50%: Moderate (4.25 Hz)
   - 100%: Fast (8.0 Hz)

2. DEPTH
   Range: 0-100%
   Default: 50%
   Use: Modulation depth
   - 0%: No modulation
   - 50%: Moderate depth
   - 100%: Maximum depth

3. FEEDBACK
   Range: 0-100%
   Default: 10%
   Use: Feedback amount (for flanger/phaser)
   - 0%: No feedback
   - 50%: Moderate feedback
   - 100%: Maximum feedback

4. MIX (SHIFT + B)
   Range: 0-100%
   Default: 50%
   Use: Dry/wet balance
   - 0%: Full dry
   - 50%: Balanced
   - 100%: Full wet

5. SYNC
   Range: 0-5
   Default: 0 (Free)
   Use: Tempo sync division
   - 0: Free (manual rate)
   - 1: 1/4
   - 2: 1/8
   - 3: 1/16
   - 4: 1/32
   - 5: 1/64

6. SHAPE
   Range: 0-3
   Default: 0 (Sine)
   Use: LFO waveform shape
   - 0: SINE
   - 1: TRIANGLE
   - 2: SQUARE
   - 3: SAW

7. STEREO
   Range: 0-100%
   Default: 50%
   Use: Stereo width
   - 0%: Mono
   - 50%: Normal stereo
   - 100%: Ultra-wide

8. COLOR
   Range: 0-100%
   Default: 50%
   Use: Tone filter
   - 0%: Dark
   - 50%: Neutral
   - 100%: Bright

9. MORPH
   Range: 0-100%
   Default: 0% (Off)
   Use: Morphing between modes
   - 0%: No morphing
   - 50%: Moderate morphing
   - 100%: Maximum morphing

USAGE TIPS
================================================================================

1. CHORUS:
   - MODE: 0 (CHORUS)
   - RATE: 25-35%
   - DEPTH: 40-60%
   - FEEDBACK: 5-15%
   - MIX: 50-70%
   - STEREO: 60-80%

2. FLANGER:
   - MODE: 1 (FLANGER)
   - RATE: 35-45%
   - DEPTH: 60-80%
   - FEEDBACK: 40-60%
   - SYNC: 2-3 (1/8 or 1/16)
   - MIX: 60-80%

3. PHASER:
   - MODE: 2 (PHASER)
   - RATE: 30-40%
   - DEPTH: 70-90%
   - FEEDBACK: 30-50%
   - SYNC: 3-4 (1/16 or 1/32)
   - MIX: 70-90%

4. AUTOPAN:
   - MODE: 5 (AUTOPAN)
   - RATE: 40-60%
   - DEPTH: 100%
   - STEREO: 100%
   - SYNC: 2-3 (1/8 or 1/16)
   - MIX: 100%

5. MORPHING:
   - Set MORPH: 50-100%
   - Switch between MODES
   - Creates smooth transitions
   - Great for live performance

================================================================================
TROUBLESHOOTING
================================================================================

Q: No effect output
A: Check MIX > 0%, ensure MODE is selected, check RATE > 0%

Q: Effect too subtle
A: Increase DEPTH (60-100%), increase MIX (70-100%)

Q: Effect too aggressive
A: Decrease DEPTH (30-50%), decrease FEEDBACK (10-30%)

Q: Tempo sync not working
A: Ensure MIDI clock is active, check SYNC parameter

Q: Rate parameter has gaps (no sound at certain positions)
A: This is fixed in latest version - RATE range is now 0.5-8 Hz (musical range)
