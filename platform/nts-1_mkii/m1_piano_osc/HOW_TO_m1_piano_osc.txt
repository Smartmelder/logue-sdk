PARAMETER 1: DECAY
  Functie: Decay time na attack fase
  Range: 0.1 - 5.0 seconden
  Default: 40% (1.0 seconde)

  0%:     Korte, staccato piano
  50%:    Medium decay (klassiek piano)
  100%:   Lange, sustaining toon

PARAMETER 2: RELEASE
  Functie: Release time na note off
  Range: 0.01 - 3.0 seconden
  Default: 25% (0.5 seconde)

  0%:     Snelle cut-off
  50%:    Natuurlijke piano release
  100%:   Lange fade-out

PARAMETER 3: BRIGHTNESS (Filter Cutoff)
  Functie: Filter cutoff frequentie
  Range: 0.1 - 1.0 (donker - helder)
  Default: 80% (bright)

  0-30%:  Dark, muffled piano
  40-60%: Warm, classic piano
  70-100%: Bright, sparkly piano

PARAMETER 4: RESONANCE (Filter Q)
  Functie: Filter resonance/emphasis
  Range: 0 - 100%
  Default: 20%

  0%:     Clean filter
  30%:    Slight emphasis
  60%+:   Strong resonance (metallic)

PARAMETER 5: VIBRATO DEPTH
  Functie: Vibrato/pitch modulation diepte
  Range: 0 - 100%
  Default: 0% (off)

  0%:     No vibrato
  10-30%: Subtle vibrato
  50%+:   Strong vibrato

PARAMETER 6: VIBRATO SPEED
  Functie: Vibrato LFO snelheid
  Range: 0.5 - 10 Hz
  Default: 50% (5 Hz)

  0%:     Slow (0.5 Hz)
  50%:    Medium (5 Hz)
  100%:   Fast (10 Hz)

PARAMETER 7: ATTACK TIME
  Functie: Attack time van envelope
  Range: 0.001 - 1.0 seconde
  Default: 1% (instant, 0.001s)

  0-10%:  Instant attack (piano)
  20-50%: Slow attack (pad-like)
  70-100%: Very slow attack

PARAMETER 8: SUSTAIN LEVEL
  Functie: Sustain level van envelope
  Range: 0 - 100%
  Default: 70%

  0%:     Pure percussive (no sustain)
  50%:    Half sustain
  100%:   Full sustain

PARAMETER 9: STEREO WIDTH
  Functie: Stereo spread/width
  Range: 0 - 100%
  Default: 50%

  0%:     Mono
  50%:    Natural width
  100%:   Maximum stereo spread

PARAMETER 10: DETUNE
  Functie: Fine tuning in cents
  Range: -100 to +100 cents
  Default: 0 cents (centered)

  -100:   Een halve toon lager
  0:      Perfect gestemd
  +100:   Een halve toon hoger

TIP 1: VELOCITY LAYERING
  Gebruik MIDI velocity voor expressie - de oscillator reageert op velocity!

TIP 2: FILTER SWEEPS
  Moduleer de BRIGHTNESS parameter met de NTS-1's LFO voor filter sweeps

TIP 3: PERCUSSIVE HITS
  Korte DECAY + korte RELEASE = percussive piano stabs

TIP 4: PAD SOUNDS
  Lange ATTACK + lange DECAY/RELEASE + vibrato = pad-achtige sounds

TIP 5: RESONANCE SWEET SPOT
  30-40% resonance geeft een mooie "body" aan het piano geluid

TIP 6: STEREO WIDTH
  60-80% width geeft een professionele, brede piano sound

TIP 7: DETUNE VOOR CHARACTER
  +5 tot +15 cents detune geeft een karaktervolle, slightly "out of tune" vibe

TIP 8: COMBINE WITH NTS-1 EFFECTS
  - Reverb: Voor natuurlijke ruimte
  - Delay: Voor rhythmic patterns
  - Chorus: Voor extra width (voorzichtig met WIDTH parameter!)
